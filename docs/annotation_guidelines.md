# FactBench 人工复核说明

每一行是“题目 + DeepSeek 的完整回答”，不评价文风，只评价回答中的可核查事实是否被可靠证据支持。

- `supported`：回答中的关键可核查断言均有可靠证据支持；允许少量不影响结论的措辞差异。
- `unsupported`：至少一个关键断言与可靠证据矛盾、无法找到支持，或虚构了来源、日期、数值、实体或引文。
- `undecidable`：问题或回答中的关键信息无法在合理时间内验证，或问题本身缺少必要条件；不能仅凭“看起来可疑”标为 unsupported。

每位评阅者应独立填写自己的 `label` 和 `evidence`。证据栏记录至少一个可复查 URL、正式出版物或权威来源名称，并用一句话说明它支持/反驳的具体断言。两位评阅者不一致时，在 `adjudicated_label` 填写裁决结论，并在 `adjudication_note` 说明理由。不要依据模型是否自称“截至某年”或“官方数据”给分；必须检查其内容。

若使用 `ai_assisted_label`，必须在报告中称其为“AI 辅助初审”，不能称为人工标注或独立评阅者。它的用途是筛出需要人工优先复看的高风险条目；最终结论应使用人工裁决列 `adjudicated_label`，或在报告中明确说明实验只进行了 AI 辅助评测。

本研究将 `unsupported` 视为需缓解的风险事件；`supported` 与 `undecidable` 分开报告，避免把“谨慎的不确定性”误算为幻觉。
