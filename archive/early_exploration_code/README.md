# Early exploration code

These scripts support early-semester explorations of GAOKAO-style multiple-choice evaluation, FactBench, manual annotation, retrieval attempts, and preliminary risk scoring. They are retained as a record of the author's individual work but are not part of the final KG-FPQ experiment pipeline.

The final, maintained experiment entry points are in [`src/`](../../src/). The archived scripts may rely on older input formats or exploratory protocols and should not be used to reproduce the reported final tables.
